## 目前实现的功能：<br>
针对信图某一房间抢各种类型(靠窗、有电脑)座位<br>
检索某个房间某个时间段的所有空座并预约<br>
加入倒计时，22:15准时抢座(为了防止系统拥堵延后10s)<br>
抢座后自动关机<br>
- - -
## 未来计划实现功能：<br>
